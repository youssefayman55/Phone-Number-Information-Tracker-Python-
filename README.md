# 📱 Phone Number Information Tracker

A Python project that extracts information from phone numbers such as:
- Country / Region
- Telecom carrier
- Validation status

## 🚀 Features
- Parse phone numbers
- Detect country using geocoder
- Identify mobile carrier
- Validate numbers

## 🛠️ Tech Stack
- Python
- phonenumbers library

## 📦 Installation
```bash
pip install phonenumbers
